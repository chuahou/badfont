FONTLOG for the GlobalFontFamily fonts

This file provides detailed information on badfont. This information should be
distributed along with badfont and any derivative works.

Basic Font Information

badfont is a Latin typeface family that supports a limited range of mostly
ASCII characters and some special characters, intended for use in programming
and terminals.

Feel free to contribute through issues and pull requests at
https://github.com/chuahou/badfont.
